# Spec: 云舱 AirCabin V1 Wireframes

## 1. Wireframe 基线
- 设计平台：Axure / Figma
- 目标设备：Android 主流 390 x 844 画板，同时兼容 360 x 800
- 栅格建议：4pt spacing system，左右安全边距 16dp
- 视觉基调：深色科技感，深蓝/黑曜灰为底，青色作为飞行状态高亮色
- 底部导航：首页 / 航班 / 统计 / 相册 / 我的

## 2. 首页 Wireframe

### 页面目标
3 秒内让用户知道当前是否有航班、当前进度、还能做什么。

### 布局结构（从上到下）
1. 顶部状态区
   - 页面标题：云舱
   - 右上角入口：消息/设置
2. 当前航班卡片
   - 航班号 + 航司
   - 起点城市/机场码 → 终点城市/机场码
   - 当前飞行阶段标签
   - 已飞时长 / 剩余时长 / ETA
   - 一条简化航线进度条
   - CTA：查看航线图
3. 快捷功能宫格（2 x 2）
   - 本次飞行统计
   - 相册整理
   - 同舱聊天室
   - 年度飞行报告
4. 年度摘要卡
   - 年度飞行次数
   - 总时长
   - 总里程
   - 最常乘坐航司
5. 历史航班入口

### 关键交互
- 点击当前航班卡片进入航班页
- 点击快捷宫格进入对应一级页面或子页面
- 无当前航班时，展示“导入登机信息”空状态按钮

### 状态说明
- 正常态：存在当前航班
- 空状态：暂无当前航班，展示导入引导
- 离线态：卡片右上显示“离线模拟”标签

## 3. 航班页 Wireframe

### 页面目标
承载“飞到哪了、还要多久、现在是什么阶段”的核心体验。

### 布局结构
1. 顶部返回栏
   - 返回
   - 标题：当前航班
   - 右上角：分享 / 更多
2. 地图容器区（页面 40% 高度）
   - 2D / 3D 切换按钮
   - 当前模拟飞机图标
   - 起点/终点标记
   - 预计航线折线
   - 地图右下角说明：当前位置为模拟估算
3. 飞行状态卡
   - 飞行阶段：起飞/爬升/巡航/下降/降落
   - 速度
   - 海拔
   - 剩余航程
   - ETA
4. 时间进度卡
   - 总时长
   - 已飞时长
   - 剩余时长
   - 起飞/预计到达时间
5. 相关入口区
   - 进入聊天室
   - 查看飞行统计
   - 查看登机信息

### 关键交互
- 地图支持拖动、缩放
- 点击 2D/3D 切换显示不同视图
- 点击飞行状态卡可展开阶段说明

### 状态说明
- 有实时修正时显示“已修正”
- 无实时修正时显示“模拟中”
- 无网时隐藏实时刷新提示，但保留本地时间轴更新

## 4. 飞行统计页 Wireframe

### 页面目标
让用户理解本次和长期飞行数据。

### 布局结构
1. 顶部切换标签
   - 本次
   - 月度
   - 季度
   - 全年
2. 核心指标卡区
   - 飞行次数
   - 总飞行时长
   - 总里程
   - 国内/国际比例
3. 图表区
   - 折线图：飞行趋势
   - 环形图：航司分布
   - 柱状图：月份/季度分布
4. 航线热力区
   - 高频航线列表
   - 城市分布
5. 年度报告 CTA
   - 生成报告卡片

### 关键交互
- Tab 切换立即刷新图表
- 图表支持 tooltip 与 drill down
- 点击某航司或航线可跳转明细列表

### 状态说明
- 首次无数据时展示“先导入一张登机牌”引导
- 年度报告已生成时展示最近一次生成时间

## 5. 相册整理页 Wireframe

### 页面目标
高效完成“筛选 - 预览 - 删除”。

### 布局结构
1. 顶部统计区
   - 可释放空间
   - 总候选数量
   - 最近扫描时间
2. 分类卡片横滑区
   - 截图
   - 大视频
   - 相似图
   - 连拍
   - 旅途当天
3. 操作工具栏
   - 全选
   - 按大小排序
   - 按时间排序
   - 进入滑动选择模式
4. 媒体网格区
   - 3 列缩略图
   - 右上角勾选状态
   - 视频显示时长角标
5. 底部操作栏
   - 预览
   - 删除
   - 取消选择

### 关键交互
- 长按进入多选模式
- 手势滑动跨格勾选
- 删除前进入预览页查看大图/视频详情
- 点击删除后先展示系统确认，再展示结果反馈

### 状态说明
- 无权限：展示权限申请引导
- 扫描中：展示进度和预计耗时
- 空状态：该分类暂无可清理内容

## 6. 聊天室页 Wireframe

### 页面目标
提供封闭、匿名、低风险的同舱交流空间。

### 布局结构
1. 顶部房间信息栏
   - 航班号
   - 舱位
   - 已验证标识
   - 在线人数
2. 公告区
   - 房间规则
   - 风险提示
3. 话题标签区
   - 拼车
   - 转机
   - 目的地建议
   - 航旅经验
4. 消息流区
   - 匿名头像
   - 昵称（如 12A 旅客）
   - 消息气泡
   - 举报按钮
5. 底部输入区
   - 输入框
   - 表情
   - 快捷话题
   - 发送按钮

### 关键交互
- 首次进入需展示规则弹窗
- 长按消息可举报/复制/拉黑发言人
- 点击话题标签可自动填充引导文案

### 状态说明
- 网络不可用：页面提示“聊天室当前不可用”
- 房间关闭：展示归档说明与结束时间
- 审核拦截：消息发送失败并给出原因反馈

## 7. 我的页 Wireframe

### 页面目标
管理飞行档案、权限和设置。

### 布局结构
1. 个人信息卡
   - 头像
   - 昵称
   - 飞行等级/徽章（预留）
2. 飞行档案入口
   - 历史航班
   - 年度飞行报告
3. 权限与隐私
   - 相册权限
   - 通知权限
   - 数据同步开关
   - 清除本地缓存
4. 设置区
   - 风格主题
   - 关于
   - 帮助与反馈

### 关键交互
- 点击飞行档案进入航班记录列表
- 点击某权限项进入系统授权说明页
- 点击清除本地缓存需二次确认

## 8. 核心流程原型页清单
建议 Axure/Figma 至少输出以下页面与关键状态：
- 首页：正常态、空状态、离线态
- 登机牌导入：拍照态、OCR 结果确认态、字段编辑态
- 航班页：2D、3D、无网模拟态
- 统计页：本次、全年、空数据态
- 相册整理页：无权限、扫描中、分类列表、多选删除确认、删除结果态
- 聊天室页：可加入、网络不可用、消息风控拦截态
- 我的页：默认态、权限管理态
